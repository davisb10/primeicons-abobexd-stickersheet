# Unofficial PrimeIcons Adobe XD Sticker Sheet

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Icons SVG files provided by [PrimeIcons](https://github.com/primefaces/primeicons)

## Usage

All icons are seperated alphabetically and then grouped with a text of the corresponding icon class name.

Icons have been converted to symbols to ease copy-pasting.
